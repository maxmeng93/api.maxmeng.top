# api.maxmeng.top

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## 安装依赖

```bash
$ pnpm install
```

## 开发

```bash
# watch mode
$ pnpm run start:dev
```

## 生产

```bash
$ pnpm run start:prod
```

## Prisma

### 修改 schema.prisma 后

```bash
$ pnpm run prisma:migrate
$ pnpm run prisma:gen
```

### 部署

```bash
$ pnpm run prisma:deploy
$ pnpm run prisma:gen
```
